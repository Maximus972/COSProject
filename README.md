# Проект простой операционной системы. 
## Технологический стек:
### 1. С (Основной ЯП); </br>2. NASM; </br>3. GRUB (чтобы не писать свой загрузчик);</br> 4. QEMU (для тестирования и запуска);</br> 5. CMAKE (для автоматизапиции сборки и запуска);

# Команды CMAKE
1. *run* - __сборка и запуск проекта__;
2. *build* - __сборка без запуска__;
3. *clean* - __удаление всех файлов и директорий, кроме исходников__;
